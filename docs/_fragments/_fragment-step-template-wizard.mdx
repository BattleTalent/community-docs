<ul>
    <li>In the <b>Unity Toolbar</b>: Go to <code>Tools -> Template Wizard</code>.</li>
    <li>Make sure the <code>inspector</code> window is shown (Template Wizard will show there).</li>
    <li>Enter a <code>new mod folder name</code>. Example: <code>YourModName</code></li>
    <li>Set the <code>Mod Type</code> to: <code>{props.modType || '<YOUR_MOD_TYPE>'}</code>.</li>
    <li>Fill in the desired <code>ItemInfoConfig</code> fields.</li>
    <li>Click the <code>Generate Template</code> button to generate the <code>{props.modType || '<YOUR_MOD_TYPE>'}</code> folder structure.</li>
</ul>
