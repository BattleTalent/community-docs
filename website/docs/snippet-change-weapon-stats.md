---
title: Change weapon stats
---

Change the stats on a weapon. In this case `damageMlp`.

## Example

```lua
self.ragdollHitInfo.hitInfo.damageMlp = 300;
```

## User Variables

| Component   | variable    |
| ----------- | ----------- |
| RagdollHitInfoObj   | `self.ragdollHitInfo` |

## FAQ

### What stats of a weapon can we change?
The following stats:

| Options |
| ------- |
| weightAmongMultiple |
| velocityMlp |
| damageMlp |
| damageThrough |
| damageCrit |
| stabMlp |
| hitMlp |
| hitRandom |
| stabDamage |
| breakDefenceMlp |
| hitBackMlp |
| knockoutFactor |
| dizzyFactor |
| stiffValue |
